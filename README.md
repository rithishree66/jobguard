# 🛡️ JobGuard AI — Fake Job Detector

> Protect yourself from fake job postings using AI-powered fraud detection.
> No API key needed. Runs 100% free.

🌐 **Live Demo:** https://jobguard-cec9.onrender.com

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🔍 Fraud Score | 0-100 score showing how fake the job is |
| ⚠️ Risk Indicators | Exact red flags found with explanations |
| ✅ Trust Signals | Genuine elements in the posting |
| 💡 Recommendation | Clear advice on what to do |
| ⚡ Instant Analysis | Results in under a second |
| 🔒 100% Private | Nothing stored, no internet needed |

---

## 🛑 Red Flags Detected

- Upfront payment / registration fee
- Requests for bank/personal info
- Unrealistic salary offers
- MLM / Pyramid schemes
- Urgency pressure tactics
- Personal email contacts (Gmail/Yahoo)
- Money transfer job duties
- Vague/generic job descriptions
- Hidden company identity

---

## 🧰 Tech Stack

- **Backend** — Python, Flask
- **Detection** — Rule-based NLP (15+ patterns)
- **Frontend** — HTML, CSS, JavaScript
- **Hosting** — Render (Free)
